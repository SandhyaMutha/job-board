# Job Board Application

A web application where users can search and explore job opportunities.

## Tech Stack

- Next.js
- React.js
- JavaScript
- CSS

## Features

- User-friendly job listing interface
- View job details
- Search jobs
- Responsive design

## Installation

Clone the repository:

git clone https://github.com/SandhyaMutha/job-board.git

Install dependencies:

npm install

Run the project:

npm run dev
